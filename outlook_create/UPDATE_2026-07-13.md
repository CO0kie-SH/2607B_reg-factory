### 关键变化

#### 1. 新增拟人鼠标模块

`common/human_mouse.py` 提供：

| 函数 | 说明 |
|---|---|
| `windmouse_path()` | 生成 WindMouse 鼠标轨迹 |
| `human_move_to()` | 使用变速轨迹移动鼠标 |
| `tremor_offsets()` | 使用 OU 过程生成自相关震颤 |
| `human_press_and_hold()` | 完整按住验证动作封装 |

原先的周期性/白噪声式抖动被替换为“路径变速 + 自相关微抖动”。

#### 2. Outlook 按住验证接入新模块

`register.py` 和 `register_outlook_standalone.py` 中的人机验证按住逻辑改为：

```python
held, passed = await _hm.human_press_and_hold(
    page,
    cx,
    cy,
    is_done=_hold_done,
    max_hold=...,
    min_hold=...,
)
```

其中 `is_done` 复用原来的“captcha iframe/元素消失”判定。

#### 3. Clash 节点探测轮换

`outlook_reg_loop.py` 新增/强化：

| 函数 | 说明 |
|---|---|
| `_probe_delay()` | 调 Clash `/delay` 探测单节点延迟 |
| `maybe_rotate_verified()` | 在候选节点中跳过超时/过慢节点，选择延迟最低者 |
| `_env_truthy_norotate()` | 解析 `OUTLOOK_NO_ROTATE` |

新增参数：

```powershell
python outlook_reg_loop.py --no-rotate
```

环境变量等价：

```powershell
$env:OUTLOOK_NO_ROTATE = "1"
```

#### 4. WebUI 脚本参数同步

`webui/scripts.py` 中 Outlook 养号/注册入口补充：

```text
--no-rotate
--sleep-when-full
```
